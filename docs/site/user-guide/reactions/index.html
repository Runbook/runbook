Reactions

Reactions are automated tasks that are called when Monitors fail. Runbook currently features Reactions for these services:

  • CloudFlare
  • Heroku
  • Commando
  • Digital Ocean
  • Amazon Web Services
  • Rackspace
  • Saltstack
  • Notifications
  • Statistics

Please visit runbook.io to learn more.
